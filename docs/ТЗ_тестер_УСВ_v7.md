## 1. ОБЩИЕ СВЕДЕНИЯ

### 1.1. Наименование системы

**Серверный тестер УСВ** (далее — Система, Тестер) — программный комплекс для испытаний устройств/систем вызова экстренных оперативных служб в испытательной лаборатории.

### 1.2. Основание для разработки

Обеспечение испытаний УСВ на соответствие требованиям **ГОСТ 33465** (редакции 2015, 2023) и **ГОСТ 33464**.

### 1.3. Назначение системы

Система предназначена для:

1. **Приём и анализ** EGTS-пакетов от тестируемого УСВ через **CMW-500**:
   - **TCP/IP канал:** УСВ → CMW-500 (WiFi) → Тестер (TCP-сервер)
   - **SMS канал:** Тестер ↔ CMW-500 (LAN, SCPI) ↔ УСВ
   - **Голосовой канал (eCall):** УСВ → CMW-500 → Тестер (тональный модем)
2. **Валидация корректности** реализации протокола (авторизация, телеметрия, траектория, профиль ускорения, eCall, обновление ПО)
3. **Протоколирование** всех этапов испытаний — каждый пакет логируется (hex + parsed)
4. **Загрузка/выгрузка данных:** работа с готовыми HEX-файлами и динамическая генерация пакетов

### 1.4. Режимы работы

| Режим | Описание |
|-------|----------|
| **Реального времени** | Тестер работает как TCP-сервер, обрабатывает пакеты по мере поступления от УСВ |
| **Офлайн (replay)** | Загрузка сохранённого лога пакетов, повторный анализ без подключения к УСВ |

### 1.7. Пример полного цикла работы

```
1. Оператор настраивает CMW-500 (IP, режим эмуляции сети)
2. Оператор запускает тестер:
   $ egts-tester start --gost 2015 --cmw 192.168.1.100
3. Ядро запускается: TCP-сервер слушает, CMW-500 подключён
4. УСВ подключается к WiFi AP CMW-500
5. УСВ регистрируется в эмулируемой сети → устанавливает TCP-соединение
6. TcpServerManager читает байты → emit("raw.packet.received")
7. PacketDispatcher получает → Pipeline.process() → emit("packet.processed")
8. SessionManager (ordered=True) обновляет FSM
9. LogManager и ScenarioManager (параллельно) обрабатывают пакет
10. Оператор запускает сценарий:
    $ egts-tester run-scenario scenarios/auth/
11. Сценарий подключается к сессии, проверяет пакеты
12. Оператор выгружает данные:
    $ egts-tester export track --format csv --output track.csv
13. Оператор останавливает сервер:
    $ egts-tester stop
```

**Ключевой принцип:** Ядро работает **всегда** — с момента `start` до `stop`. Сценарий — **опциональная надстройка** для автоматизации проверки. Без сценария ядро:
- Принимает пакеты ✅
- Отправляет RESPONSE ✅
- Обрабатывает дубликаты PID ✅
- Ведёт FSM ✅
- Логирует всё ✅
- Транзакции PID↔RPID ✅

Сценарий добавляет только: **проверку ожидаемых пакетов**, **capture переменных**, **автоматическую отправку команд по шагам**, **отчёт PASS/FAIL**.

## 2. АРХИТЕКТУРА

### 2.1. Принципы

1. **Единое ядро (Core Engine)** — вся бизнес-логика в одном модуле. CLI и GUI — **тонкие обёртки** над ядром
2. **EventBus для слабой связанности** — компоненты общаются через шину событий, не знают друг о друге
3. **Чёткое разделение ответственности** — каждый модуль/класс делает **одну** вещь
4. **НЕ смешивать логику** — парсинг ≠ валидация ≠ бизнес-логика ≠ логирование ≠ интерфейс
5. **Простота** — можно применять обоснованные абстракции которые приводят к улучшению кода

### 2.2. Core Engine (единое ядро)

```python
class CoreEngine:
    """Единое ядро системы. Координирует компоненты через EventBus."""

    def __init__(self, config: Config):
        self.bus = EventBus()
        self.tcp_mgr = TcpServerManager(bus=self.bus, port=config.tcp_port)
        self.cmw_ctrl = Cmw500Controller(bus=self.bus, ip=config.cmw_ip)
        self.session_mgr = SessionManager(bus=self.bus, gost_version=config.gost_version)
        self.packet_dispatcher = PacketDispatcher(bus=self.bus, session_mgr=self.session_mgr)
        self.command_dispatcher = CommandDispatcher(bus=self.bus, session_mgr=self.session_mgr)
        self.scenario_mgr = ScenarioManager(bus=self.bus, session_mgr=self.session_mgr)
        self.logger_mgr = LogManager(bus=self.bus)

    async def start(self):
        await self.tcp_mgr.start()
        await self.cmw_ctrl.connect()
        await self.bus.emit("server.started", {})

    async def stop(self):
        await self.tcp_mgr.stop()
        await self.cmw_ctrl.disconnect()
        await self.bus.emit("server.stopped", {})
```

Реализация с поддержкой ordered и parallel хендлеров:

```python
import asyncio
from dataclasses import dataclass
from typing import Callable, Awaitable

@dataclass
class Event:
    name: str
    data: dict

class EventBus:
    """Async-совместимая шина событий с поддержкой ordered и parallel хендлеров."""

    def __init__(self):
        self._handlers: dict[str, list[Callable]] = {}
        self._ordered_handlers: dict[str, list[Callable]] = {}

    def on(self, event_name: str, handler: Callable, ordered: bool = False):
        if ordered:
            self._ordered_handlers.setdefault(event_name, []).append(handler)
        else:
            self._handlers.setdefault(event_name, []).append(handler)

    def off(self, event_name: str, handler: Callable):
        if handler in self._handlers.get(event_name, []):
            self._handlers[event_name].remove(handler)
        if handler in self._ordered_handlers.get(event_name, []):
            self._ordered_handlers[event_name].remove(handler)

    async def emit(self, event_name: str, data: dict):
        # 1. Ordered-хендлеры (FSM, критичные) — последовательно
        for handler in self._ordered_handlers.get(event_name, []):
            if asyncio.iscoroutinefunction(handler):
                await handler(data)
            else:
                handler(data)

        # 2. Parallel-хендлеры (логирование, сценарии) — параллельно
        tasks = []
        for handler in self._handlers.get(event_name, []):
            if asyncio.iscoroutinefunction(handler):
                tasks.append(asyncio.create_task(handler(data)))
            else:
                handler(data)

        if tasks:
            await asyncio.gather(*tasks, return_exceptions=True)
```

#### 2.6.2. Cmw500Controller с очередью команд

```python
class Cmw500Controller:
    def __init__(self, bus: EventBus, ip: str):
        self.bus = bus
        self._ip = ip
        self._queue: asyncio.Queue = asyncio.Queue()
        self._worker: asyncio.Task | None = None
        self._connected = False

    async def connect(self):
        try:
            self._worker = asyncio.create_task(self._worker_loop())
            self._connected = True
            await self.bus.emit("cmw.connected", {"ip": self._ip})
        except Exception as e:
            await self.bus.emit("cmw.error", {"error": str(e), "command": "connect"})
            raise

    async def disconnect(self):
        self._connected = False
        if self._worker:
            self._worker.cancel()
        await self.bus.emit("cmw.disconnected", {})

    async def execute(self, command: CmwCommand, *args) -> str:
        if not self._connected:
            raise ConnectionError("CMW-500 not connected")
        future: asyncio.Future = asyncio.get_event_loop().create_future()
        await self._queue.put((command, args, future))
        return await future

    async def send_sms(self, pdu: bytes) -> bool:
        result = await self.execute(SEND_SMS, pdu.hex())
        return "OK" in result

    async def get_imei(self) -> str:
        return await self.execute(GET_IMEI)

    async def _worker_loop(self):
        while self._connected:
            try:
                command, args, future = await asyncio.wait_for(
                    self._queue.get(), timeout=1.0
                )
                result = await self._execute_with_retry(command, *args)
                future.set_result(result)
            except asyncio.TimeoutError:
                continue
            except Exception as e:
                if 'future' in locals():
                    future.set_exception(e)
            finally:
                self._queue.task_done()

    async def _execute_with_retry(self, command: CmwCommand, *args) -> str:
        last_error = None
        for attempt in range(command.retry_count):
            try:
                scpi = command.format(*args)
                result = await asyncio.wait_for(
                    self._send_scpi(scpi), timeout=command.timeout
                )
                return result
            except Exception as e:
                last_error = e
                await asyncio.sleep(command.retry_delay * (2 ** attempt))
        raise last_error
```

### 2.7. PacketDispatcher и CommandDispatcher

#### 2.7.1. PacketDispatcher

```python
class PacketDispatcher:
    """Единственный компонент, который знает о Pipeline."""

    def __init__(self, bus: EventBus, session_mgr: SessionManager):
        self.bus = bus
        self.session_mgr = session_mgr
        self.pipeline = self._build_pipeline()
        bus.on("raw.packet.received", self._on_raw_packet)

    def _build_pipeline(self) -> PacketPipeline:
        pipeline = PacketPipeline()
        pipeline.add(CrcValidationMiddleware())      # 1. CRC
        pipeline.add(ParseMiddleware())              # 2. Парсинг
        pipeline.add(DuplicateDetectionMiddleware(self.session_mgr))  # 3. Дубликаты
        pipeline.add(TransactionMatchMiddleware(self.session_mgr))    # 4. Транзакции
        pipeline.add(EventEmitMiddleware(self.bus))  # 5. Публикация
        return pipeline

    async def _on_raw_packet(self, data: dict):
        ctx = await self.pipeline.process(
            raw=data["raw"], channel=data["channel"], connection_id=data["connection_id"]
        )
        await self.bus.emit("packet.processed", {
            "ctx": ctx, "connection_id": data["connection_id"]
        })
```

#### 2.7.2. CommandDispatcher

```python
class CommandDispatcher:
    def __init__(self, bus: EventBus, session_mgr: SessionManager):
        self.bus = bus
        self.session_mgr = session_mgr
        bus.on("command.send", self._on_command)

    async def _on_command(self, data: dict):
        conn = self.session_mgr.get_connection(data["connection_id"])
        if not conn:
            await self.bus.emit("command.error", {
                "error": f"Connection {data['connection_id']} not found",
                "step_name": data.get("step_name")
            })
            return

        try:
            conn.writer.write(data["packet_bytes"])
            await conn.writer.drain()

            if data.get("step_name"):
                conn.transaction_mgr.register(
                    pid=data.get("pid"), rn=data.get("rn"),
                    step_name=data["step_name"], timeout=data.get("timeout", 30)
                )

            await self.bus.emit("command.sent", {
                "connection_id": data["connection_id"],
                "step_name": data.get("step_name"),
                "packet_bytes": data["packet_bytes"]
            })
        except Exception as e:
            await self.bus.emit("command.error", {
                "error": str(e), "step_name": data.get("step_name")
            })
```

```python
    remote_port: int

    writer: asyncio.StreamWriter | None = None

    next_rn: int = 0

    # LRU-кэш для дубликатов
    _seen_pids: OrderedDict = field(default_factory=OrderedDict)
    MAX_SEEN_PIDS: int = 65536

    def add_pid_response(self, pid: int, response: bytes):
        if pid in self._seen_pids:
            self._seen_pids.move_to_end(pid)
        self._seen_pids[pid] = response
        if len(self._seen_pids) > self.MAX_SEEN_PIDS:
            self._seen_pids.popitem(last=False)

    def get_response(self, pid: int) -> bytes | None:
        if pid in self._seen_pids:
            self._seen_pids.move_to_end(pid)
            return self._seen_pids[pid]
        return None

    @property
    def usv_id(self) -> str:
        return str(self.tid) if self.tid is not None else self.connection_id
```

#### 3.2.2. UsvStateMachine

```python
class UsvStateMachine:
    STATES = ["DISCONNECTED", "CONNECTED", "AUTHENTICATING", "CONFIGURING",
              "AUTHORIZED", "RUNNING", "ERROR"]

    def __init__(self, usv_id: str):
        self.usv_id = usv_id
        self.state = "DISCONNECTED"

    def on_packet(self, parsed: dict) -> str | None:
        service = parsed.get("service")
        result_code = parsed.get("result_code")
        tid = parsed.get("TID", -1)

        # Основные переходы
        if self.state == "CONNECTED" and service == 1:
            self.state = "AUTHENTICATING"
            return "process_auth"

        if self.state == "AUTHENTICATING" and result_code == 153:
            self.state = "CONFIGURING"
            return "start_configuring"

        if self.state == "CONFIGURING" and tid > 0:
            self.state = "AUTHENTICATING"
            return "process_auth"

        if self.state == "AUTHENTICATING" and result_code == 0:
            self.state = "AUTHORIZED"
            return "notify_authorized"

        if self.state == "AUTHORIZED" and service == 2:
            self.state = "RUNNING"
            return "notify_running"

        if parsed.get("_protocol_error"):
            self.state = "ERROR"
            return "notify_error"

        # Неожиданные пакеты — допустимые переходы
        if self.state == "CONNECTED" and service == 2:
            self.state = "RUNNING"
            return "notify_running"

        if self.state in ("RUNNING", "AUTHORIZED") and service == 1:
            self.state = "AUTHENTICATING"
            return "process_auth"

        return None  # Игнорируем, SessionManager логирует WARNING

    def on_timeout(self) -> str | None:
        if self.state in ("AUTHENTICATING", "CONFIGURING", "RUNNING"):
            self.state = "DISCONNECTED"
            return "close_connection"
        return None

    def on_disconnect(self) -> None:
        self.state = "DISCONNECTED"
```

### 3.3. PacketPipeline

```python
@dataclass
class PacketContext:
    raw: bytes
    parsed: dict | None = None
    crc8_valid: bool = False
    crc16_valid: bool = False
    crc_valid: bool = False
    is_duplicate: bool = False
    errors: list[str] = field(default_factory=list)
    terminated: bool = False
    channel: str = "tcp"
    connection_id: str = ""

class PacketPipeline:
    def __init__(self):
        self._middlewares: list[PacketMiddleware] = []

    def add(self, middleware: PacketMiddleware):
        self._middlewares.append(middleware)

    async def process(self, raw: bytes, channel: str, connection_id: str) -> PacketContext:
        ctx = PacketContext(raw=raw, channel=channel, connection_id=connection_id)
        for mw in self._middlewares:
            try:
                await mw.process(ctx)
            except Exception as e:
                ctx.errors.append(f"{mw.__class__.__name__}: {str(e)}")
                ctx.terminated = True
                break
            if ctx.terminated:
                break
        return ctx
```

```python
@dataclass
class Variable:
    value: any
    created_at: float = field(default_factory=time.monotonic)
    ttl: float | None = None

    @property
    def is_expired(self) -> bool:
        if self.ttl is None:
            return False

        self.history: list[dict] = []

    def get(self, name: str) -> any | None:
        var = self._variables.get(name)
        if var and not var.is_expired:
            return var.value
        return None

    def substitute(self, template: str) -> str:

        return re.sub(
            r"\{\{(\w+)\}\}",
            lambda m: str(self.get(m.group(1)) or ""),
            template

            if conn.fsm and conn.fsm.state not in ("DISCONNECTED", "ERROR"):
                self.current_connection_id = conn_id
```

#### 3.5.2. ExpectStep

```python
class ExpectStep:
    def __init__(self, config: dict):
        self.name = config["name"]
        self.match = config.get("match", {})
        self.capture = config.get("capture", {})
        self.timeout = config.get("timeout", 30)
        self.in_response_to = config.get("in_response_to")
        self._event: asyncio.Event | None = None
        self._matched_packet: dict | None = None
        self._context: ScenarioContext | None = None

    async def execute(self, context: ScenarioContext) -> dict:
        self._context = context
        self._event = asyncio.Event()
        self._matched_packet = None

        handler = self._on_packet
        context.bus.on("packet.processed", handler)

        self._disconnect_event = asyncio.Event()
        def on_disconnect(data):
            if data.get("usv_id") == context.current_connection_id:
                self._disconnect_event.set()
        context.bus.on("connection.changed", on_disconnect)

        try:
            done, _ = await asyncio.wait(
                [
                    asyncio.create_task(self._event.wait()),
                    asyncio.create_task(self._disconnect_event.wait()),
                    asyncio.create_task(asyncio.sleep(self.timeout)),
                ],
                return_when=asyncio.FIRST_COMPLETED,
            )

            if self._event.is_set():
                return {"status": "PASS", "step": self.name, "data": self._matched_packet}
            if self._disconnect_event.is_set():
                return {"status": "ERROR", "step": self.name, "error": "USV disconnected"}
            return {"status": "TIMEOUT", "step": self.name}
        finally:
            context.bus.off("packet.processed", handler)
            context.bus.off("connection.changed", on_disconnect)

    def _on_packet(self, data: dict):
        if self._event and self._event.is_set():
            return
        ctx = data["ctx"]
        if not ctx.parsed:
            return

        if self.in_response_to and self._context:
            txn = self._context.get(f"_txn_{self.in_response_to}")
            if not txn:
                return
            rpid = ctx.parsed.get("RPID")
            crn = ctx.parsed.get("CRN")
            if rpid != txn.get("pid") and crn != txn.get("rn"):
                return

        if self._matches(ctx.parsed):
            if self._context:
                for var_name, field_path in self.capture.items():
                    value = self._get_nested(ctx.parsed, field_path)
                    if value is not None:
                        self._context.set(var_name, value)
            self._matched_packet = ctx.parsed
            self._event.set()

    def _matches(self, parsed: dict | None) -> bool:
        if not parsed or not self.match:
            return False
        for key, expected in self.match.items():
            actual = self._get_nested(parsed, key)
            if actual is None:
                return False
            if isinstance(expected, dict):
                if not self._match_criterion(actual, expected):
                    return False
            elif isinstance(expected, str) and expected.startswith("^"):
                import re
                if not re.match(expected, str(actual)):
                    return False
            else:
                if actual != expected:
                    return False
        return True

    def _match_criterion(self, actual, criterion: dict) -> bool:
        criterion_type = criterion.get("type", "any")
        value = str(actual) if criterion_type == "string" else actual
        if "range" in criterion:
            low, high = criterion["range"]
            if not (low <= value <= high):
                return False
        if "regex" in criterion:
            import re
            if not re.match(criterion["regex"], str(value)):
                return False
        if "length" in criterion and criterion_type == "string":
            if len(str(value)) != criterion["length"]:
                return False
        return True

    @staticmethod
    def _get_nested(data: dict, path: str):
        import re
        parts = re.split(r"[\.\[\]]+", path.strip())
        parts = [p for p in parts if p]
        current = data
        for part in parts:
            if current is None:
                return None
            if isinstance(current, dict):
                current = current.get(part)
            elif isinstance(current, (list, tuple)):
                try:
                    current = current[int(part)]
                except (IndexError, ValueError):
                    return None
            else:
                return None
        return current
```

#### 3.5.3. SendStep

```python
class SendStep:
    def __init__(self, config: dict):
        self.name = config["name"]
        self.from_file = config.get("from_file")
        self.build = config.get("build")
        self.channel = config.get("channel", "tcp")
        self.timeout = config.get("timeout", 30)

    async def execute(self, context: ScenarioContext) -> dict:
        conn_id = context.current_connection_id
        if not conn_id:
            conn_id = context._resolve_connection_id()
            if not conn_id:
                return {"status": "ERROR", "step": self.name, "error": "No active connection"}

        packet_bytes = await self._build_packet(context)

        protocol = create_protocol(context.gost_version)
        packet = protocol.parse_packet(packet_bytes)
        pid = packet.get("PID")
        records = packet.get("records", [])
        rn = records[0].get("RN") if records else None

        context.set(f"_txn_{self.name}", {"pid": pid, "rn": rn})

        future: asyncio.Future = asyncio.get_event_loop().create_future()
        def on_sent(data):
            if data.get("step_name") == self.name:
                future.set_result(data)
        context.bus.on("command.sent", on_sent)

        try:
            await context.bus.emit("command.send", {
                "connection_id": conn_id, "packet_bytes": packet_bytes,
                "step_name": self.name, "pid": pid, "rn": rn, "timeout": self.timeout,
            })
            await asyncio.wait_for(future, timeout=self.timeout)
            return {"status": "PASS", "step": self.name}
        except asyncio.TimeoutError:
            return {"status": "TIMEOUT", "step": self.name}
        finally:
            context.bus.off("command.sent", on_sent)

    async def _build_packet(self, context: ScenarioContext) -> bytes:
        if self.from_file:
            path = Path(self.from_file)
            if not path.is_absolute():
                path = Path(context.scenario_dir) / self.from_file
            return bytes.fromhex(path.read_text().strip())
        if self.build:
            protocol = create_protocol(context.gost_version)
            build_config = json.loads(context.substitute(json.dumps(self.build)))
            return self._build_from_template(protocol, build_config)
        raise ValueError("SendStep требует 'from_file' или 'build'")

    def _build_from_template(self, protocol, template: dict) -> bytes:
        subrecord = template.get("subrecord", "")
        fields = template.get("fields", {})
        if subrecord == "RECORD_RESPONSE":
            return protocol.build_record_response(
                crn=fields.get("CRN", 0), rst=fields.get("RST", 0)
            )
        elif subrecord == "RESULT_CODE":
            return protocol.build_result_code(
                result=fields.get("result", 0), tid=fields.get("TID")
            )
        else:
            raise ValueError(f"Unsupported subrecord: {subrecord}")
```

```python
class ReplaySource:
    def __init__(self, log_path: str):
        self.log = self._load_log(log_path)

    def _load_log(self, log_path: str) -> dict:
        with open(log_path, "r") as f:

    async def replay(self, bus: EventBus, pipeline: PacketPipeline | None = None,

        for packet in self.log.get("packets", []):
            if packet.get("direction") != "rx":
                continue

            parsed = packet.get("parsed")

                    raw=raw, parsed=parsed,

                    crc16_valid=packet.get("validation", {}).get("crc16") == "ok",
                    crc_valid=(packet.get("validation", {}).get("crc8") == "ok"
                               and packet.get("validation", {}).get("crc16") == "ok"),
                    is_duplicate=False,
                    channel=packet.get("channel", "tcp"),
                    connection_id=conn_id,
                )

            await bus.emit("packet.processed", {"ctx": ctx, "connection_id": conn_id})
            await asyncio.sleep(0.01)
```

| 3 | **Траектория** | TRACK_DATA, последовательность точек |
| 4 | **Профиль ускорения** | ACCEL_DATA, разрешение ≤ 0.01G, ASI15 |
| 5 | **eCall** | RAW_MSD_DATA (ASN.1), голосовой вызов, AL-ACK |
| 6 | **Обновление ПО** | SERVICE_PART_DATA, SERVICE_FULL_DATA, ODH |
| 7 | **Команды и параметры** | COMMAND_DATA, запрос/установка параметров |
| 8 | **Режим тестирования** | TEST_MODE_ON/OFF, проверочные пакеты |
| 9 | **SMS-канал** | Приём/отправка EGTS через SMS, конкатенация |
| 10 | **Пассивный режим** | Конфигурирование при TID=0, SMS с учётными данными, повторная авторизация |

## 5. CLI ИНТЕРФЕЙС

**CLI — тонкая обёртка над Core Engine.** Не содержит бизнес-логики, только: разбор аргументов, вызов методов Core Engine, форматирование вывода.

### 5.1. Команды

```bash
# Запуск сервера
egts-tester start --port 3001 --gost 2015 --cmw 192.168.1.100

# Replay-режим (офлайн анализ)
egts-tester replay logs/session_351234567890123.json --scenario scenarios/auth/

# Запуск сценария
egts-tester run-scenario scenarios/auth/ --output report.json

# Пакетный запуск
egts-tester batch --scenario auth --scenario telemetry --output report.json

# Статус
egts-tester status

# Выгрузка данных
egts-tester export track --format csv --output track.csv

# Интерактивный монитор
egts-tester monitor

# Стоп
egts-tester stop
```

### 5.3. Программный API

```python
from egts_tester import CoreEngine, Config

async def main():
    config = Config(gost_version="2015", tcp_port=3001, cmw_ip="192.168.1.100")
    engine = CoreEngine(config)
    await engine.start()
    result = await engine.run_scenario("scenarios/auth/")
    print(result.summary())
    await engine.stop()
```
