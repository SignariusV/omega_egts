# Известные проблемы

## EGTS-протокол

- **SRT коды в `SubrecordType`** — значения перечисления не совпадают с номерами из ГОСТ.
  Реальные коды берутся из `gost.md` Таблица 40: SRT 20 = ACCEL_DATA, SRT 63 = TRACK_DATA.
  Перечисление `SubrecordType` используется только для читаемости, парсеры регистрируются
  по числовому `srt`.

- **COMMAND_CODES 0x0501/0x0502** — коды `EGTS_UNIT_MIC_LEVEL` и `EGTS_UNIT_SPK_LEVEL`
  добавлены на основе контекста ТЗ, требуют верификации по ГОСТ 33465-2015 Таблица 34.

## CMW-500

- **Файл учётных данных на Windows** — `attrib +h` мешает последующей записи.
  Администратор должен вручную ограничить ACL (Properties → Security → Advanced).

## Тесты

- **GUI-тесты** — `tests/gui/test_main.py` и связанные файлы требуют PyQt-окружения,
  пропускаются в CI.

## Производительность

- **LogManager** — буферизация записей может приводить к задержкам при высокой нагрузке.
  Настройка `flush_interval` доступна, но требует тонкой настройки под конкретную среду.
