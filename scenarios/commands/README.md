# Команды конфигурирования (EGTS_COMMANDS_SERVICE)

## Описание
Платформа отправляет команды конфигурирования на УСВ через EGTS_COMMANDS_SERVICE (ГОСТ 6.7.3).

## Схема обмена

```
┌──────────┐                                    ┌─────┐
│ПЛАТФОРМА │                                    │ УСВ │
└────┬─────┘                                    └──┬──┘
     │ 1. COMMAND_DATA  → команда (72 байт)       │
     │ 2. PT_RESPONSE   ← ответ УСВ (29 байт)     │
     │ 3. CT_COMCONF    ← подтверждение (49 байт) │
     │                                            │
     │  ✅ Команда выполнена                      │
```

## Шаги

| # | Пакет | Направление | Размер | Описание |
|---|-------|-------------|--------|----------|
| 1 | EGTS_SR_COMMAND_DATA | Платформа → УСВ | 72 байт | Команда конфигурирования |
| 2 | EGTS_PT_RESPONSE | УСВ → Платформа | 29 байт | Ответ транспортного уровня |
| 3 | CT_COMCONF | УСВ → Платформа | 49 байт | Подтверждение выполнения |

## Пакеты

- `packets/platform/command_data.hex` — команда конфигурирования
- `packets/usv/pt_response.hex` — ответ транспортного уровня
- `packets/usv/comconf_command.hex` — подтверждение выполнения
