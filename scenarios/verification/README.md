# Верификация (SMS)

## Описание
Первоначальная настройка УСВ через SMS-канал перед подключением по TCP/IP.

## Схема обмена

```
┌──────────┐      SMS-КАНАЛ           ┌─────┐
│ПЛАТФОРМА │                          │ УСВ │
└────┬─────┘                          └──┬──┘
     │ 1. GPRS_APN     → настройки APN  │
     │ 2. COMCONF      ← подтверждение  │
     │ 3. SERVER_ADDR  → адрес сервера  │
     │ 4. COMCONF      ← подтверждение  │
     │ 5. UNIT_ID      → запрос ID      │
     │ 6. COMCONF      ← ответ с ID     │
     │                                  │
     │  ✅ Верификация завершена        │
```

## Шаги

| # | Пакет | Направление | Размер | Описание |
|---|-------|-------------|--------|----------|
| 1 | EGTS_GPRS_APN | Платформа → УСВ | 46 байт | Настройки APN (internet) |
| 2 | CT_COMCONF | УСВ → Платформа | 33 байт | Подтверждение APN (CID=0) |
| 3 | EGTS_SERVER_ADDRESS | Платформа → УСВ | 55 байт | Адрес сервера (200.20.2.171:9090) |
| 4 | CT_COMCONF | УСВ → Платформа | 33 байт | Подтверждение адреса (CID=1) |
| 5 | EGTS_UNIT_ID | Платформа → УСВ | 42 байт | Запрос UNIT_ID |
| 6 | CT_COMCONF | УСВ → Платформа | 33 байт | Ответ с UNIT_ID (CID=2) |

## Пакеты

- `packets/platform/gprs_apn.hex` — настройки APN
- `packets/platform/server_address.hex` — адрес сервера
- `packets/platform/unit_id.hex` — запрос UNIT_ID
- `packets/usv/comconf_*.hex` — подтверждения от УСВ
